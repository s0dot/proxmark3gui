iCLASS / Picopass

Dump files are saved in the client's working directory (.bin / .json / .eml).

HF Write

Write blocks to 13.56 MHz cards. Identify the card first so you know its type.

MIFARE Classic — write block
hf mf wrbl
⚠ Writing block 0 with invalid data can brick a Magic Gen2 card.
Magic card (Gen1a) — set UID
hf mf csetuid — magic Gen1a only
Ultralight / NTAG — write page
hf mfu wrbl
